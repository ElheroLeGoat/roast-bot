Command Help

Don't understand any commands or want to know additional information about them? Then this is the place to look. Listed below is the help given what you do rb!commandName help but with more updated info and some beta commands info.

rb!user help:

rb!user has 2 ways that it can be used. The first being rb!user which will return stats about you such as when your account was created, your presence(online, offline etc), user id, and your current game. The second way is rb!user @user. This way returns the stats of whoever you tagged. The returned stats are the same.

Example 1:

USER: rb!user
Roast-Bot: Account created on.... current game.... user id.... and so forth.

Example 2:

USER: rb!user @Roast-Bot
Roast-Bot: Account created on.... current game.... user id.... and so forth.

Note: You can also see multiple people's stats by doing rb!user @user1 @user2 and so forth depending on how many people you want to tag.

rb!server help:

rb!server tells you information about your server. The information that it provides is: Number of people in server, Server name, the date you joined, and the date the server was created.

Example:

USER: rb!server
Roast-Bot: Server Name....
Date the server was created....
Date you joined....
Number of people in the server....

rb!server will still work even if you put 1 or more space after the "server". It will not work however if you put 1 or more spaces and then a character that isn't a space.

rb!roast help:

rb!roast has 3 different ways that it can be used. The three ways are:
rb!roast
rb!roast #roastNumber
rb!roast @USER

rb!roast generates a random roast. It's as simple as that. All you have to do is rb!roast

Example:
USER: rb!roast
Roast-Bot: You must've been born at a pound because your a son of a bitch.
Roast #99 :roast_circle:

rb!roast #roastNumber is a way to return a specific roast. At the end of every roast it will say "Roast #..." the number is what number of roast it is.

Example:
USER: rb!roast #99
Roast-Bot: You must've been born at a pound because your a son of a bitch.
Roast #99

rb!roast @USER is the last way to use rb!roast. The way that this command works is you roast a person in your server.

Example:
USER: rb!roast @Roast-Bot#0168
Roast-Bot: @Roast-Bot#0168, Some babies were dropped on their heads but you were clearly thrown at a wall.
Roast #41

rb!meme help:

rb!meme has 2 ways that it can be used. These 2 ways are as follows:
rb!meme
rb!meme #memeNumber

rb!meme will generate a random meme. You can put rb!meme , notice the space and it will also generate a random meme the same as rb!meme with no space would.

Example:
USER: rb!meme
Roast-Bot: Meme #someNumber a meme

rb!meme #memeNumber is almost as simple as rb!meme, the only difference is that memeNumber, is which meme you want to be sent.

Example:
USER:rb!meme #12
Roast-Bot: Meme #12meme picture

rb!clear help:

rb!clear NUMBER is simple enough to use, NUMBER is the amount of messages that you want to be deleted.

Example:

USER: rb!clear 10
Roast-Bot: Cleared 10 messages.

rb!clear NUMBER will still work if you add 2 or more spaces after rb!clear.

Example:

USER: rb!clear 1
Roast-Bot: Cleared 1 message.

Another incorrect way is to send just rb!clear with no number. Roast-Bot will send a message back saying "Incorrect usage of rb!clear, please provide how many messages you want to be deleted. The correct usage is rb!clear NUMBER. "

If the bot or user doesn't have the correct permissions to use rb!clear it will return a warning message. For user not having the Manage Messages permission Roast-Bot will return "Looks like you dont have the permissions to do that :( ". For Roast-Bot not having the correct permissions to delete messages the message "Roast-Bot needs to be given Manage Messages permissions to use this command :( " will be sent.

Another error of rb!clear NUMBER is trying to delete too many messages. Discord doesn't allow 100 or more messages to be deleted in one command. You can though send multiple rb!clear commands to delete more than 100 messages combined. If the user doesn't know about the message limit this error: The max number of messages you can delete is 100 :( will be sent.


If your still having trouble with rb!clear NUMBER please join the support server and ask in #roast-bot-help.
